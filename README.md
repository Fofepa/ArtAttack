# Art Attack

## Installazione 

Nelle release è presente il file JAR con il source code.
Scaricare il file JAR, entrare nella cartella dove è stato scaricato ed aprire il terminale, scrivere nel terminale il seguente comando ``` java -jar ArtAttack.jar``` . 

## Requisiti 
- JDK 21


## Informazioni aggiuntive
Il gioco è stato sviluppato e testato su ambiente Windows. Su altri sistemi operativi potrebbero presentarsi problemi o imperfezioni nella grafica.
